Detour: Why Not Just Use Linear Attention?

Katharopoulos et al. 2020. A simpler model, but with a critical flaw.

Linear attention

Replace softmax attention with a simple running sum:

$M_t = M_{t-1} + \phi(k_t)\,v_t^\top$

Output: $o_t = M_t\,\phi(q_t)$

This is fast and parallel. But it only appends to memory. It never edits or erases old entries.

DeltaNet (for comparison)

$S_t = S_{t-1}(I - \beta_t k_t k_t^\top) + \beta_t\,v_t\,k_t^\top$

Erases along $k_t$ before writing. Old memories are actively corrected.

Head-to-head

Linear AttentionDeltaNet
WriteAppend new entryErase old, write new
Old memoriesNever touchedActively edited
Retrieval errorGrows with seq lenStays bounded

Analogy: Linear attention is a wiki where you can only append pages. DeltaNet is a wiki where you can also edit and overwrite existing pages.

Linear attention is the starting point that DeltaNet improves on. The erase-before-write mechanism is what makes DeltaNet's memory useful over long sequences.
