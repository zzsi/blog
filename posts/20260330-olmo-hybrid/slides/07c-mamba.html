Detour: The Other Lineage (Mamba / SSM)

Gu & Dao, 2024. The main competing family of recurrent layers.

State space model recurrence

A discrete-time linear recurrence with input-dependent parameters (selective SSM):

$h_t = \bar{A}_t \, h_{t-1} + \bar{B}_t \, x_t$
$y_t = C_t \, h_t$

$h_t \in \mathbb{R}^{n}$ is the hidden state. $\bar{A}_t, \bar{B}_t, C_t$ are all input-dependent (Mamba's key innovation over S4, which used fixed $A, B$).

Like DeltaNet, this is a fixed-size recurrent state updated per token. Like DeltaNet, training uses chunk-parallel algorithms.
