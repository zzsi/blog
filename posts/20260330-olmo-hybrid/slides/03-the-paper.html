The Paper: OLMo Hybrid (Allen AI, March 2026)

What they did

Took OLMo 3 7B (a pure transformer) and replaced 75% of attention layers with Gated DeltaNet recurrent layers. Trained on the same data with the same recipe.

A controlled experiment: the only variable is the architecture.

Model: 7B parameters, 6T training tokens

Architecture: 3:1 GDN-to-attention, interleaved

Infra: 512 GPUs (H100 → B200 migration mid-run)

Fully open: weights, code, data, training logs

Main results

OLMo 3 (Transformer)OLMo Hybrid
Tokens to match MMLU ~6T ~3T (49% fewer)
Per-layer state 512 MiB (full attn) 1 MiB (GDN), 485x smaller
RULER 64K 70.9 85.0 (+14.1)

After mid-training

Beats OLMo 3 on every eval domain: math +1.4%, code +0.8%, STEM +3.6%, non-STEM +3.1%, GenQA +2.3%.
