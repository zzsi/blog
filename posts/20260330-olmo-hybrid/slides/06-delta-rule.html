Step 1: The Delta Rule (1960)

Widrow & Hoff, 1960. The simplest gradient-based learning rule.

Setting

A linear predictor with weight $w$, input $x_t$, target $y_t$. The prediction is $\hat{y}_t = w_{t-1}^\top x_t$. Here $t$ indexes training examples.

Gradient descent on squared error gives the update rule:

$w_t = w_{t-1} + \eta \underbrace{(y_t - w_{t-1}^\top x_t)}_{\delta_t\text{ = prediction error}} \cdot\; x_t$

Correct the weight in proportion to the error. This is an online update: one example arrives, weights change immediately.

A pattern that keeps reappearing

MEMORY A stored state ($w_{t-1}$)

READ Retrieve a prediction ($w_{t-1}^\top x_t$)

WRITE Error-correcting update ($\delta_t \cdot x_t$)

This structure (memory, read, write) is exactly what shows up in DeltaNet 65 years later, with $w$ replaced by a matrix and $x$ replaced by key/value pairs.
