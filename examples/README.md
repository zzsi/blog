# MCP Examples

These examples support the draft blog post on building and deploying MCP for proprietary data.

- `mcp-readonly-starter`: single-tenant, read-only tools
- `mcp-multitenant-saas`: tenant-aware authz and routing
- `mcp-write-with-approval`: write tools behind approval workflow
- `mcp-onprem-bridge`: pull-based on-prem bridge agent
- `mcp-compliance-audit`: strict policy and audit-first deployment
