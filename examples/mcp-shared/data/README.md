# Synthetic Demo Data

This folder contains synthetic datasets for MCP demo scenarios.

## invoices.csv

Fields:
- `invoice_id`
- `account_id`
- `status` (`paid`, `open`, `overdue`, `void`)
- `amount`
- `currency`
- `due_date` (ISO date)
- `updated_at` (ISO timestamp)

The data is synthetic and safe for public demos.
